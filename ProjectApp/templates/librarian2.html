How Does It Work?

  • Each reader can borrow a book for a period of 14 days
  • Each reader can hold up to 3 books at any given moment
  • If the reader has 3 books, the loan request cannot be approved
  • Order requests are kept in the system for 3 days from the moment the book is returned
  • If a copy is ordered by a reader, borrow requests of other readers cannot be approved for this copy
  • You can add a new book or an additional copy of an existing book to the system in the 'Add New Book' tab
